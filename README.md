## 文字教程
1. 浏览器F12查看`favorite/?`接口
2. 将数据复制到`data_favorite.json`
3. 运行`main`方法
4. 打开`target/classes`目录下的`index.html`

## 操作逻辑
1. 点击、空格,暂停或播放视频
2. 方向下与方向右，播放下一个视频
3. 方向上与方向左，播放上一个视频
4. 回车，旋转视频90度
